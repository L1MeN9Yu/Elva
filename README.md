# Elva

The [zstd](https://github.com/facebook/zstd.git) and [brotli](https://github.com/google/brotli.git) wrapper.

## Contribution

1. `brew bundle` to install `pre-commit` , `swiftformat` , `swiftlint`
2. `pre-commit install` to install git hook.
