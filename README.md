# Elva

The `Compression` Kit.

## Modules

### ZSTD

The [zstd](https://github.com/facebook/zstd.git) Swift wrapper.

### Brotli

[brotli](https://github.com/google/brotli.git) Swift wrapper.

### LZ4

[LZ4](https://github.com/lz4/lz4.git) Swift wrapper.
